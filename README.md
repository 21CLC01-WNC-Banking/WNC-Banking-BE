# Steps to Deploy the Project

- Install Docker and Docker Compose.
- Ensure ports 3000, 3636, and 8080 are free.
- Navigate to the "deploy" directory inside the src folder.
- Run the command to start Docker Compose:
```bash
$ docker compose up -d
```
